================================================================================
 TROUBLESHOOTING
================================================================================

Problem: "ModuleNotFoundError: No module named 'librosa'"
Solution: Install libraries using pip install command above

Problem: Training is very slow
Solution: 
- Close other programs
- Run overnight
- Consider using Google Colab with GPU

Problem: Out of memory error
Solution: 
- Reduce BATCH_SIZE in train_transfer_hubert.py (line 23)
- Change from 8 to 4 or 2

Problem: CUDA out of memory
Solution:
- Reduce batch size
- Or set device to CPU in training script
